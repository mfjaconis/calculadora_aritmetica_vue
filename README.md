# Calculadora Aritmética

Este projeto é uma calculadora aritmética simples e responsiva desenvolvida utilizando Vue.js e Bootstrap. O objetivo é permitir que o usuário insira dois números, escolha uma operação matemática e visualize o resultado em tempo real.

## Funcionalidades

- **Operações Suportadas**:
  - Soma
  - Subtração
  - Multiplicação
  - Divisão (com tratamento para divisão por zero)
- **Atualização em tempo real**: O resultado é calculado automaticamente ao alterar os valores ou a operação.
- **Responsividade**: Utiliza o sistema de grid do Bootstrap para adaptar o layout a diferentes tamanhos de tela.
- **Estilização Dinâmica**: A cor do texto muda de acordo com a operação selecionada.

## Tecnologias Utilizadas

- [Vue.js 3](https://vuejs.org/)
- [Bootstrap 5](https://getbootstrap.com/)
- HTML, CSS

## Como Executar o Projeto

1. **Clone o repositório**:
   ```bash
   git clone <url-do-repositorio>
   cd <nome-do-diretorio>
   ```
2. **Instale as dependências**:
   ```bash
    npm install
   ```
3. **Execute o servidor de desenvolvimento**:
    ```bash
   npm run dev
   ```
4. Abra no navegador: Acesse http://localhost:5173 para visualizar o projeto.
